<p>빈즈란 특정 작업을 독립적으로 수행하는 컴포넌트를 말함
컴포넌트란 특정 기능을 수행하는 부품을 의미하므로 프로그래밍 관점에서 컴포넌트란 정해진 기능을 수행하는 클래스를 의미
자바빈즈 = jsp 문서와 연동하기 위한 자바 클래스</p>
<h3 id="자바빈즈-액션-태그">자바빈즈 액션 태그</h3>
<blockquote>
<p><a href="">jsp:useBean</a> : 빈즈 객체를 생성하기 위한 액션 태그
 <a href="">jsp:setProperty</a> : 빈즈 객체의 프로퍼티에 값을 지정하기 위한 액션 태그
 <a href="">jsp:getProperty</a> : 빈즈 객체의 프로퍼티의 값을 추출하기 위한 액션 태그</p>
</blockquote>
<blockquote>
<p>&lt;jsp:useBean id=&quot;빈즈 객체 참조변수&quot; class=&quot;자바빈즈 클래스 이름&quot; scope=&quot;범위&quot; /&gt;</p>
</blockquote>
<ul>
<li>id : 생성할 빈즈 객체를 참조하기 위한 변수</li>
<li>class : 패키지 이름을 포함한 자바빈즈 클래스의 이름</li>
<li>scope : 빈즈 객체가 사용되는 영역으로 page(기본값), request, session, application 중 하나를 지정 </li>
</ul>
<blockquote>
<p>&lt;jsp:setProperty name=&quot;자바빈즈 참조변수&quot; property=&quot;멤버변수 이름&quot; value=&quot;값&quot;&gt;</p>
</blockquote>
<ul>
<li>name  : 멤버변수의 값을 지정할 빈즈 객체의 참조 변수</li>
<li>property : 값을 지정할 빈즈 객체의 멤버 변수의 이름</li>
<li>value : 임의의 값을 빈즈 객체의 멤버 변수에 저장<h5 id="form-요소의-파라미터-이름과-빈즈-객체의-멤버-변수-이름이-같다면">form 요소의 파라미터 이름과 빈즈 객체의 멤버 변수 이름이 같다면?</h5>
getParameter() 메서드 없이 파라미터 값을 멤버 변수에 저장할 수 잇음<blockquote>
<pre><code>&lt;input type=&quot;text&quot; name=&quot;depart&quot; size=20 &gt; 
submit할 때 그 값이 파라미터의 값으로 입력됨.</code></pre></blockquote>
</li>
<li>파라미터로 전달된 값이 존재하는데 value 속성으로 다른 값을 지정하면 전달된 파라미터 값은 무시되고 value 속성으로 지정된 값이 저장됨<h6 id="param-속성">param 속성</h6>
파라미터 이름을 지정할 때 사용 다만 value랑 같이 사용 못함.</li>
</ul>
<blockquote>
<p>&lt;jsp:getProperty name=&quot;빈즈&quot; property=&quot;출력할 멤버 변수&quot;&gt;</p>
</blockquote>
<ul>
<li>name : 멤버변수의 값을 출력할 빈즈 객체의 참조 변수</li>
<li>property : 출력할 멤버변수의 이름을 지정</li>
</ul>
