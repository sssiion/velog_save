<h3 id="http-프로토콜의-동작">HTTP 프로토콜의 동작</h3>
<blockquote>
<p>HTTP 프로토콜은 클라이언트의 요청을 개별적이고 독립적으로 간주합니다
HTTP 프로토콜은 상태 정보를 유지하지 않습니다.</p>
</blockquote>
<h4 id="쿠키-생성">쿠키 생성</h4>
<blockquote>
<p>Cookie 쿠키객체 = new Cookie(&quot;쿠키이름&quot;, &quot;쿠키값&quot;);
response.addCookie(쿠키객체);</p>
</blockquote>
<blockquote>
<p>getCookies() : 클라이언트에서 전달된 모든 쿠키를 추출해 쿠키 배열로 반환
getName() : 쿠키의 이름 추출
getValue() : 쿠키의 값을 추출</p>
</blockquote>
<p>같은 이름의 다른 쿠키를 넣으면 쿠키 값을 변경할 수 있음</p>
<h5 id="쿠키-유효시간-지정">쿠키 유효시간 지정</h5>
<blockquote>
<p>setMaxAge(int expiry) : 쿠키의 유효시간을 촏ㄴ위로 지정</p>
</blockquote>
<blockquote>
<p>Cookie 쿠키객체 = new Cookie(&quot;쿠키이름&quot;, &quot;쿠키값&quot;);
쿠키객체.setMaxAge(시간);
response.addCookie(쿠키객체);</p>
</blockquote>
<p>시간을 0으로 지정하고 쿠키객체를 add하면 쿠키를 제거할수도 있음.</p>
<h4 id="세션-생성">세션 생성</h4>
<ul>
<li>쿠키는 클라이언트에 상태 정보를 유지하는 반면 서버에 상태정보를 유지함.</li>
<li>서블릿에서 세션을ㄹ 사용하려면 HttpSession 객체를 생성해야 함.<blockquote>
<p>application =pageContext.getServletContext();
config = pageContext.getSerlvetConfig();
session = pageContext.getSession();
out = pageContext.getOut();</p>
</blockquote>
</li>
</ul>
<h5 id="세션-정보-추출">세션 정보 추출</h5>
<blockquote>
<p>isNew() : 세션이 처음으로 시작되었으면 true 반환
getID() : 세션 ID를 반환
getCreationTime() : 세션이 시작된 시각 반환
getLastAccessedTime() :  세션에 접근한 마지막 시각을 반환.</p>
</blockquote>
<h5 id="세션-속성-지정">세션 속성 지정</h5>
<blockquote>
<p>setAttribute(String name , String value) : 세션 속성 name의 값으로 value 지정
getAttribute(String name) :이름이 name인 세션 속성의 값을 반환
removeAttribute(String name) : 이름이 name인 세션 속성을 제거</p>
</blockquote>
<blockquote>
<p>HttpSession session = request.getSession();  // 세션 객체 생성, 가져오기</p>
</blockquote>
<h5 id="세션-유효-시간-지정-메서드">세션 유효 시간 지정 메서드</h5>
<blockquote>
<p>setMaxInactiveInterval(int t) : 세션의 유효시간을 초 단위로 지정
getMaxInactiveIntervale() : 세션에 지정된 유효시간을 초 단위로 반환</p>
</blockquote>
<h5 id="세션의-종료">세션의 종료</h5>
<blockquote>
<p>invalidate() : 세션을 종료함.</p>
</blockquote>
<h6 id="세션-예시">세션 예시</h6>
<p>UserVO.java(클래스) : 멤버변수 선언
LoginForm.jsp : 로그인 입력 받는 문서
UserDAO.java(클래스) : DataSource 객체를 생성해서 서블릿으로부터 전달받은 인자로 값을 찾음
UserLogin.java(클래스) : @webServlet(&quot;/login&quot;)으로 jsp쪽에서 연결되어잇는 함수, jsp에 값을 받고 DAO에도 값을 보내서 수행한 값을 돌려받고 로그인 기능은 여기있음.</p>
